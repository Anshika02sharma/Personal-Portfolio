Project

RSA Encryption, Decryption, and Digital Signature

This project implements RSA encryption, decryption, and digital signature functionality in Java. It encrypts a plaintext message into numeric chunks and decrypts them back to the original message. The digital signature is generated by hashing the message and encrypting it with the private key, then verified with the public key.
